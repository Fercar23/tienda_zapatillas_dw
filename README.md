# Tienda Zapatillas – Proyecto DWES

Proyecto desarrollado en PHP siguiendo una arquitectura de 3 capas (presentación,
lógica de negocio y acceso a datos).

## Requisitos
- XAMPP
- PHP 8.x
- MySQL
- Navegador web

## Instalación
1. Clonar o descargar el repositorio.
2. Copiar la carpeta del proyecto dentro del directorio `htdocs`.
3. Importar la base de datos MySQL incluida en la carpeta `sql`.
4. Iniciar los servicios Apache y MySQL desde XAMPP.
5. Acceder a la aplicación mediante:
   http://localhost/tienda_zapatillas
